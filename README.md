# Shalala Homes

A responsive property management website built with Angular.

Visitors can browse rental listings, learn about available services, company info and contact Shalala Homes through WhatsApp or SMS. Property owners can securely log in to create, edit, and delete rental listings with images using Supabase.

## Built With

- Angular
- SCSS
- Supabase

## Run Locally

```bash
npm install
npm start
